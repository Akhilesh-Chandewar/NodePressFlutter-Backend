# Nodepress Flutter

## Installation

Follow these steps to set up and run the project locally:

1. **Clone the repository:**

    ```bash
    git clone <repository_url>
    ```

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Run Prisma migrations:**

    ```bash
    npx prisma migrate dev
    ```

4. **Start the development server:**

    ```bash
    npm start
    ```
